# 📦 Warehouse 2: Factory Logs System

Профессиональная система учета производственных логов для склада. Проект реализует полный цикл (Fullstack): от работы с базой данных до генерации отчетности.

## 🚀 Основные функции
*   **Контроль реестра:** Система позволяет вносить изменения только по существующим в базе номерам рулонов.
*   **Строгая логика статусов:** Блокировка повторного завершения (SET) для рулонов, которые уже закрыты.
*   **История логов:** Каждое действие оператора сохраняется как отдельная запись с временной меткой.
*   **Экспорт в Excel:** Генерация кастомизированных отчетов с фильтрацией по датам.
*   **Облачное хранение:** Интеграция с MongoDB Atlas для доступа к данным из любой точки.

## 🛠 Технологический стек
*   **Backend:** Node.js, Express.js
*   **Database:** MongoDB (Mongoose), MongoDB Atlas
*   **Frontend:** HTML5, CSS3, JavaScript (Fetch API)
*   **Reporting:** ExcelJS

## 📋 Как запустить
1. Склонируйте репозиторий.
2. Установите зависимости: `npm install`.
3. Создайте файл `.env` и добавьте вашу строку подключения `MONGO_URI`.
4. Запустите сервер: `node index.js`.
5. Откройте `index.html` в браузере.
